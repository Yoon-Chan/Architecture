## BookSearchApp
 + 책 검색 기능 앱 구현

### search(검색 부분)

+ 사용 기술

  + Retrofit

  + OkHttp

  + Gson

  + repository

  + ViewModel, ViewModelFactory
  + RecyclerView

+ 상단 EditTextView를 사용하여 입력을 받을 때마다 검색한 결과를 보여주는 기능 구현
+ Retrofit을 이용하여 카카오 API와 연결하여 책의 정보를 받아와 해당 책에 맞는 이름을 리사이클러 뷰를 통해 보여주도록 하는 기능
